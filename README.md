## About
Aplikasi Web Service yang dibuat menggunakan laravel 8 dengan penerapan JWT untuk login

## Response List
422 - Validator Error
201 - Register Success
200 - Auth & Process Success
401 - Unauthotized
409 - Register Vailed

## Route
POST /login
POST /register
POST /logout
POST /task
GET /task
PUT /task/{id}
DELETE /task
